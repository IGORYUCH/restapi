POST
/restapi/users/create
json={'username':username, 'password':password}

result: 201, {}
Note: Username format should be like your.name@somemail.com

POST
/restapi/users/login
json={'username':email, 'password':password}

result: 200, {"access_token": token}

GET
/restapi/tasks
headers = {Authorization: token}

result: 200, {"tasks_id": [...]}
Note: Authorization header format should be: " JWT " + acces_token

GET
/restapi/tasks/<id>
headers = {Authorization: token}

result: 200, {"task"={"title":.., "description":.., "complete":.., "complete_date":..}}

PUT
/restapi/tasks/<id>/complete
headers = {Authorization: token}

result: 200, {}

DELETE
/restapi/tasks/<id>/delete
headers = {Authorization: token}

result: 200, {}

POST
/restapi/tasks/create
headers = {Authorization: token}
json={"title":.., "description":.., "complete_date":..}

result: 201, {}
Note: Date format should be ISO (yyyy-mm-dd)
